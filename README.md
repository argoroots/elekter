API and website to get hourly electricity price.
API query parameters:
- plan - Grid service plan: V1, V2, V2k, V4 or V5.

API response is in following format:
```
  [
    year,
    month,
    day,
    hour,
    price,
    gridFee,
    renewableTax,
    excise
  ]
```
